# Parishay HealthHub – Clinic Management Backend

This is a **Spring Boot REST API** for a **clinic management system**.  
It covers **employees, attendance, patients, appointments, visits, billing and payroll**.

---

## 1. Tech Stack

- **Language:** Java (e.g. 17 – update if needed)
- **Framework:** Spring Boot
- **Build Tool:** Maven
- **Database:** MySQL / H2 (update according to your project)
- **Tools:** STS / IntelliJ, Git, GitHub, Postman

---

## 2. Main Features (Planned / Implemented)

- Employee & attendance management  
- Patient registration and profile management  
- Doctor & staff management  
- Appointments booking and visit tracking  
- Basic billing / payroll (planned)

---

## 3. How to Run

1. **Clone the repository**

   ```bash
   git clone https://github.com/tassimamustafa/parishay-healthhub-clinic-backend.git
   cd parishay-healthhub-clinic-backend
   ```

2. **Configure database** in `src/main/resources/application.properties` (update URL, username, password).

3. **Build & run**

   ```bash
   mvn clean install
   mvn spring-boot:run
   ```

4. Application will start at:

   ```text
   http://localhost:8080
   ```
